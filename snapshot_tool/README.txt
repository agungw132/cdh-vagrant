Virtualbox cluster snapshot tool

Installation:
1. Extract files from archive (snapshot.sh, VBoxManage.bat) into your Vagrant folder (for example ~/vagrant/centos6.4)
2. Start MSYS shell(Windows) or bash shell (Linux), navigate to that folder
3. Execute "chmod +x snapshot.sh"
4. You are ready to go! Type "./snapshot.sh" to see help on usage

# Note that on Windows you must sync time on your machines explicitly after snapshot taking and restoring manually
